# 星辰科技智能边缘盒子 Pro V2 发布公告

> 发布部门：智能硬件事业部
> 发布日期：2026年5月18日

## 产品概述

星辰科技第二代智能边缘计算盒子 **StarEdge Pro V2** 正式发布，面向工业场景的实时AI推理需求设计。

## 核心规格

| 指标 | 参数 |
|------|------|
| AI算力 | 32 TOPS (INT8) |
| 内存 | 16GB LPDDR5 |
| 存储 | 256GB NVMe SSD |
| 功耗 | 15W（典型） |
| 接口 | 2x GigE, 4x USB 3.0, 1x HDMI 2.0 |
| 工作温度 | -20°C ~ 65°C |
| 操作系统 | StarOS 3.0（基于Ubuntu 22.04 LTS） |

## 支持模型

- YOLOv8 / YOLOv11 目标检测
- ResNet / MobileNet 图像分类
- Whisper 语音识别
- 自定义 ONNX 模型

## 上市时间

即日起接受预订，2026年6月15日正式发货。首批售价 **人民币 4,999 元/台**。
