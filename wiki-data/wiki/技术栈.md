# 技术栈

## 技术选型总览

| 层次 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 后端框架 | FastAPI | 最新 | 高性能 Python Web 框架 |
| 服务器 | Uvicorn | 最新 | ASGI 服务器 |
| 大模型 | DeepSeek (deepseek-chat) | - | 通过 OpenAI 兼容接口调用 |
| Embedding | BAAI/bge-small-zh | - | 中文语义向量模型 |
| 向量数据库 | ChromaDB | 最新 | 嵌入式向量数据库 |
| 前端 | 原生 HTML/CSS/JS | - | 轻量级聊天界面 |
| 编程语言 | Python 3.13+ | 3.13 | 项目主力语言 |

## 后端框架：FastAPI

[FastAPI](https://fastapi.tiangolo.com/) 是一个现代、高性能的 Python Web 框架，专为构建 API 而设计。

**为什么选择 FastAPI？**
- **高性能**：与 Node.js 和 Go 相当的性能
- **自动生成文档**：自动生成 Swagger UI 和 ReDoc 文档
- **类型安全**：基于 Pydantic 的类型校验
- **异步支持**：原生支持 async/await
- **依赖注入**：清晰的依赖管理机制

## 大模型：DeepSeek

系统通过 OpenAI 兼容接口调用 [DeepSeek](https://platform.deepseek.com/) 的 `deepseek-chat` 模型。

**配置方式（.env）：**
```
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-xxxx
OPENAI_MODEL=deepseek-chat
OPENAI_BASE_URL=https://api.deepseek.com/v1
```

**DeepSeek 优势：**
- 中文能力优秀，特别适合法律等专业领域
- 性价比高，API 费用远低于 GPT-4
- 支持 128K 上下文窗口
- OpenAI 兼容接口，迁移成本极低

## Embedding 模型：bge-small-zh

[BAAI/bge-small-zh](https://huggingface.co/BAAI/bge-small-zh) 是北京智源人工智能研究院推出的中文 Embedding 模型。

**技术指标：**
- 输出维度：512
- 模型大小：约 33MB
- 支持语义相似度计算
- 针对中文优化

## 向量数据库：ChromaDB

[ChromaDB](https://www.trychroma.com/) 是一个开源的嵌入式向量数据库。

**核心特性：**
- **嵌入式部署**：无需单独启动数据库服务
- **持久化存储**：数据保存在本地磁盘
- **多种索引**：支持余弦距离、L2 距离等
- **Python 原生**：与 Python 生态无缝集成
- **轻量级**：适合中小规模知识库

### 系统配置

| 配置项 | 配置文件 | 说明 |
|--------|---------|------|
| LLM 配置 | `.env` | API Key、模型选择、Base URL |
| 系统配置 | `core/config.py` | 端口、分块参数、缓存策略 |
| 文档目录 | `.env` | Wiki 数据目录、向量库目录 |

### .env 完整示例

```env
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-your-key-here
OPENAI_MODEL=deepseek-chat
OPENAI_BASE_URL=https://api.deepseek.com/v1
CHROMA_PERSIST_DIR=./chroma_db
WIKI_DATA_DIR=../wiki-data
HOST=0.0.0.0
PORT=8000
```

