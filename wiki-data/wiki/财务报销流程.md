# 财务报销流程

## 总则

为规范公司费用报销管理，控制运营成本，确保财务合规，特制定本流程。所有员工因公产生的费用均需按本流程报销。

## 费用分类与标准

### 差旅费用

| 费用类别 | 普通员工 | 部门主管 | 总监及以上 |
|---------|---------|---------|-----------|
| 机票 | 经济舱 | 经济舱 | 商务舱 |
| 高铁 | 二等座 | 一等座 | 一等座/商务座 |
| 酒店 | ≤ 400 元/晚 | ≤ 600 元/晚 | ≤ 1000 元/晚 |
| 市内交通 | 实报实销 | 实报实销 | 实报实销 |
| 餐补 | 80 元/天 | 100 元/天 | 150 元/天 |

出差住宿费、交通费超出标准的部分需在报销时备注说明，由主管特批。

### 业务招待费

| 招待对象 | 人均标准 | 陪同人数限制 |
|---------|---------|------------|
| 重要客户 | ≤ 500 元/人 | ≤ 3 人陪同 |
| 普通客户 | ≤ 300 元/人 | ≤ 2 人陪同 |
| 合作伙伴 | ≤ 200 元/人 | ≤ 2 人陪同 |

### 办公用品

- 单价 ≤ 500 元：部门主管审批
- 单价 500-5000 元：行政部审批
- 单价 > 5000 元：分管副总审批
- 固定资产（单价 ≥ 2000 元且使用期限 ≥ 1 年）需行政部登记入册

### 培训费用

| 类型 | 年度限额 | 审批人 |
|------|---------|--------|
| 外部培训 | 3000 元/人/年 | 部门主管 |
| 考证报销 | 2000 元/人/年 | 人力资源部 |
| 学历提升 | 5000 元/人/年 | 总经理 |

## 报销流程

### 标准报销流程

```
报销人提单 → 部门主管审批 → 财务初审 → 财务经理复核 → 出纳付款 → 归档
```

### 各环节说明

1. **报销人提单**
   - 登录 OA 系统 → 费用报销 → 新建报销单
   - 填写费用类型、金额、事由、项目归属
   - 上传发票照片或扫描件（每张发票清晰可见）
   - 提交后系统自动推送审批

2. **部门主管审批**
   - 核实费用真实性、合理性
   - 确认费用是否在预算范围内
   - 审批时限：2 个工作日

3. **财务初审**
   - 检查发票合规性（真伪、抬头、税号、金额）
   - 核对报销标准是否超标
   - 审批时限：3 个工作日

4. **财务经理复核**
   - 抽检大额报销（单笔 ≥ 5000 元）
   - 确认预算执行情况
   - 审批时限：1 个工作日

5. **出纳付款**
   - 每周三、周五集中付款
   - 付款方式默认打入工资卡
   - 紧急报销（单笔 ≤ 2000 元）可走加急通道

### 加急报销

满足以下任一条件可申请加急：
- 因公垫付金额超过个人月工资 50%
- 出差归来后 3 个工作日内提交的差旅报销
- 因突发情况产生的紧急费用

加急报销在审批单标注"加急"字样，各环节时限减半。

## 发票要求

### 合规发票要素

| 要素 | 要求 |
|------|------|
| 发票抬头 | 公司全称（不得缩写） |
| 纳税人识别号 | 统一社会信用代码 18 位 |
| 发票内容 | 必须与实际消费一致，不得笼统写"办公用品" |
| 发票日期 | 不得早于出差/消费日期 30 天，不得晚于报销提交日 |
| 发票印章 | 必须加盖销售方发票专用章，清晰可见 |

### 电子发票

- 需同时上传 PDF 原件和截图
- 电子发票重复报销检测：报销前需在 OA 系统中查询
- 同一张电子发票不得重复报销，违者按公司制度处理

## 报销时限

| 费用类型 | 报销期限 |
|---------|---------|
| 差旅费用 | 出差返回后 15 个工作日内 |
| 日常费用 | 费用发生后 30 天内 |
| 预付款 | 预付款核销期限 60 天 |
| 年度费用 | 当年 12 月 25 日前提交，逾期转入次年 |

超过报销期限的费用原则上不予报销，特殊情况需总经理特批。

## 借款管理

### 因公借款

- 出差借款：按出差天数 × 预估费用，上限 5000 元
- 项目借款：凭合同或立项文件，按需申请
- 临时借款：单次上限 3000 元

### 核销要求

借款应在公务完成后 15 个工作日内凭发票核销，逾期未核销的从当月工资中扣回。

## 常见违规情形

1. **虚假报销**：伪造、变造发票或虚假填报 → 解除劳动合同 + 追回款项
2. **超额报销**：超标部分不予报销，差额由个人承担
3. **重复报销**：同一费用重复申请 → 书面警告 + 追回款项
4. **逾期报销**：超期费用不予报销（特批除外）
5. **混开发票**：个人消费与公务消费开在同一张发票上 → 不予报销
