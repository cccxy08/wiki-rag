# 合同管理制度

## 总则

为规范公司合同管理，防范合同法律风险，保障公司合法权益，特制定本制度。本制度适用于公司对外签订的所有合同、协议及具有法律约束力的文件。

## 合同分类

### 按金额分级

| 级别 | 金额范围 | 审批层级 | 签署权限 |
|------|---------|---------|---------|
| A 类 | ≥ 100 万元 | 董事会 | 董事长/总经理 |
| B 类 | 10-100 万元 | 总经理办公会 | 总经理 |
| C 类 | 1-10 万元 | 分管副总 | 部门总监 |
| D 类 | < 1 万元 | 部门主管 | 部门主管 |

### 按类型分类

| 类型 | 说明 | 模板编号 |
|------|------|---------|
| 采购合同 | 物资、设备、服务采购 | CT-PUR-001 |
| 销售合同 | 产品、服务销售 | CT-SAL-001 |
| 技术合同 | 技术开发、转让、咨询 | CT-TEC-001 |
| 保密协议 | NDA、数据保护协议 | CT-NDA-001 |
| 劳动合同 | 员工劳动合同、劳务协议 | CT-HR-001 |
| 租赁合同 | 房屋、设备租赁 | CT-LEA-001 |
| 合作协议 | 战略合作、联营协议 | CT-COOP-001 |

## 合同审批流程

### 标准流程

```
发起人起草 → 法务初审 → 财务审核（涉及金额）→ 技术审核（涉及技术）→ 分管领导审批 → 授权签署 → 归档
```

### 各环节职责

| 环节 | 负责部门 | 审核要点 |
|------|---------|---------|
| 起草 | 业务部门 | 合同基本信息、商务条款、交付标准 |
| 法审 | 法务部 | 法律条款合规性、风险条款识别、违约责任设定 |
| 财务 | 财务部 | 付款方式、发票类型、税务合规 |
| 技术 | 技术部 | 技术指标验收标准、知识产权归属 |
| 审批 | 管理层 | 综合决策 |

### 审批时限

| 环节 | 时限 |
|------|------|
| 法务初审 | 2 个工作日 |
| 财务审核 | 1 个工作日 |
| 技术审核 | 2 个工作日 |
| 管理层审批 | 3 个工作日 |

> 紧急合同可走"加急通道"，各环节时限减半。

## 合同要素规范

### 必备条款

每份合同必须包含以下条款：

1. **当事人信息**：名称、地址、统一社会信用代码、联系方式
2. **标的条款**：产品/服务的名称、规格、数量、质量标准
3. **价款条款**：金额、币种、支付方式、发票类型
4. **履行条款**：交付时间、地点、方式、验收标准
5. **违约责任**：违约情形、违约金比例（不超过合同总金额 30%）
6. **争议解决**：选择仲裁或诉讼，明确管辖地
7. **保密条款**：保密范围、期限（通常不低于 3 年）
8. **知识产权条款**：知识产权归属、使用许可范围

### 禁止条款

以下条款在合同中应**严格避免**出现：
- 无限连带责任担保
- 无上限赔偿条款
- 管辖权对公司不利的约定
- 违反法律法规强制性规定的条款

## 合同签署

### 签署方式

| 方式 | 适用范围 | 法律效力 |
|------|---------|---------|
| 纸质签署 | A 类合同、需面签的合同 | 完整法律效力 |
| 电子签章 | B/C/D 类合同、常规协议 | 与纸质签署同等效力（《电子签名法》） |
| 线上确认 | 订单、补充协议 | 需保留完整聊天记录或确认记录 |

### 授权管理

- 合同签署人须经公司书面授权
- 授权书应明确授权范围、金额上限和有效期限
- 授权到期后自动失效，需重新办理
- 严禁越权签署、倒签合同

## 合同归档

### 归档要求

- 合同签署后 **5 个工作日内** 完成归档
- 纸质合同归档至档案室，电子版上传至合同管理系统
- 归档材料包括：合同正本、附件、审批单、授权书
- 归档编号规则：`YYYY-类型-序号`（如：2026-PUR-001）

### 保管期限

| 合同类型 | 保管期限 |
|---------|---------|
| A 类合同 | 永久 |
| B 类合同 | 合同终止后 15 年 |
| C/D 类合同 | 合同终止后 10 年 |
| 保密协议 | 协议终止后 5 年 |

## 合同变更与终止

### 变更流程

合同变更视同新合同审批，需重新走完整审批流程。变更内容需以书面补充协议形式确认，口头变更无效。

### 终止情形

1. **正常终止**：合同履行完毕，权利义务终结
2. **协商终止**：双方达成一致，签署终止协议
3. **单方解除**：按合同约定的解除条款执行，或法定解除
4. **违约解除**：对方根本违约，经催告后在合理期限内仍未履行

## 合同台账管理

每月 5 日前，法务部需更新合同台账，内容包括：
- 合同编号、名称、类型
- 合同对方名称
- 合同金额
- 签署日期、生效日期、到期日期
- 履行状态（正常/变更/终止/纠纷）
- 归档位置
