# 项目概述

## 简介

CodeXFiles（Wiki-RAG 企业知识问答系统）是一个基于 **Karpathy LLM Wiki 范式** 与 **RAG (Retrieval-Augmented Generation)** 双引擎驱动的企业级知识管理平台。该项目由芙蓉律师事务所 AI 团队主导开发，旨在解决企业内部知识碎片化、检索效率低、知识沉淀难等核心痛点。

## 核心目标

- **知识统一管理**：将散落在文档、邮件、聊天记录中的企业知识结构化沉淀
- **智能问答**：支持自然语言查询，即时获取精准答案
- **知识复利**：每次问答自动评估质量，高价值问答自动沉淀到 Wiki
- **双引擎兜底**：Wiki 精准命中优先，RAG 语义检索兜底，确保零遗漏

## 技术架构概览


## 核心特性

### 1. 双引擎架构
- **Wiki 引擎**：基于 LLM 自动维护的结构化知识库，支持页面的创建、更新、交叉引用和过期管理
- **RAG 引擎**：基于 Embedding + ChromaDB 的语义检索，对非结构化文档进行智能分块和向量化

### 2. 知识复利
当 RAG 回答质量评分 ≥ 7 分时，系统自动将问答对转化为 Wiki 页面，实现知识的自动化沉淀和积累。

### 3. 三种查询模式
- **Auto 模式**：根据问题长度和复杂度自动选择最优引擎
- **Pipeline 模式**：固定流水线（Wiki → RAG 兜底）
- **Agent 模式**：ReAct Agent 自主决策，可自行决定使用哪个工具

### 4. 健康检查与维护
- Wiki Lint：自动检测孤立页面、过期内容、缺少引用的页面
- 缓存系统：查询缓存 + 评分缓存，提升重复查询效率

## 适用场景

- 企业内部知识库管理
- 法律文档智能检索
- 项目文档自动化维护
- 新人培训知识问答
